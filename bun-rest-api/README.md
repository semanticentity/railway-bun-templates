## API Endpoints

- `GET /api/items?page=1&limit=10` - List items (paginated)
- `POST /api/items` - Create item
- `GET /api/items/:id` - Get item
- `PUT /api/items/:id` - Update item
- `DELETE /api/items/:id` - Delete item
